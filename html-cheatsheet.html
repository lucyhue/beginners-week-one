Lucy's HTML Cheatsheet

Heading tags

There are five possible heading tags, from h1 to h5.

This is how you create the tags:

  • <h1>heading1</h1>
  • <h2>heading2</h1>
  • <h3>heading3</h1>
  • <h4>heading4</h1>
  • <h5>heading5</h1>

Lists

You can create ordered lists (numbered) or unordered lists (bullet points).

Ordered lists

This is how you create an ordered list:

First, you need to introduce the tag using <ol>.

Next, you introduce each list item using <li>

  1. First list item
  2. Second list item
  3. Third list item

Unordered lists

This is how you create an unordered list:

First, you introduce the tag using <ul>.

Next, you introduce each list item using <li>, just like the ordered lists.

  • First list item
  • Second list item
  • Third list item

Bold and italics

You can make text bold or italics using HTML tags.

Bold

To make text bold, you use the tag <strong>, with the closing tag </strong>.

For example: The text is now bold.

Italics

To make text italic, you use the tag <em>, with the closing tag </em>.

For example: The text is now italics.
